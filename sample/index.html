Ember Bootstrap

A set of UI elements styled using Twitter Boostrap toolkit to use with Ember.js

Modal panes

Ember.ModalPane.popup({
  heading: "Sample modal pane",
  message: "Sample message",
  primary: "OK",
  secondary: "Cancel"
});
